# Mantis-Bug-Tracker
Trabajo de final de Grado medio para el IES San Mamede en el ciclo de SMIR
Este trabajo de Mantis bug Tracker funcionará como una Guia para la instalación y uso del programa Web de gestión de incidencias como herramienta fundamental para muchos proyectos de desarrollo que requieran trabajo en equipo dispersos.
Toda documentación y código se verá añadida a un archivo completo con explicaciones y solución de errores según puedan estos suceder en una instalación rutinaria.
